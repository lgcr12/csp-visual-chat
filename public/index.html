配置聊天和配音

1. 聊天 API

本地预览可以直接聊天，但不会调用真实模型。要让角色按 Skill 正常回复，选择 OpenAI Compatible 或 Claude Messages API。

  • OpenAI Compatible：填 Base URL、Model、API Key。
  • Claude：填 Claude API Key，Base URL 通常可留空。
  • API Key 只保存在本机浏览器 localStorage。

2. ACGN 配音

推荐优先选择 TTS-Online 官方 API。官方模式需要 token，网页兼容模式可以不填 token，适合先测试。

  • Voice Model / ACGN 语言：日语填 JP，中文填 ZH。
  • 中文回复自动转日语配音：勾上后中文台词会发给 ACGN 自动翻译。
  • 官方 API：token 必填，也可以直接粘贴 www.ttson.cn/?token=xxx 链接。
  • 网页兼容：token 可不填，不填走本机游客额度。充值链接要先用 acgn.ttson.cn?token=xxx 打开一次同步额度。

3. 语音延迟

ACGN 需要生成音频后才能播放，长回复会更慢。现在已启用按句分段生成，第一句会先播放，后面的句子继续排队。
